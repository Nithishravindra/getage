# getage

Simple CLI to know your age. 

## Install

```
npm i -g get-agee
```

## Usage

```
âžœ  ~ get-agee help
   
      To know your age:
      $ get-agee
  
      To set your date of birth:
      $ get-agee YYYY-MM-DD (ISO Format)
  
      To remove previously set date of birth:
      $ get-agee clear
  
      To output version:
      $ get-agee version|--version|-v
  
      To output this help message:
      $ get-agee help|--help|-h
```
